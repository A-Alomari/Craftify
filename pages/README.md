# Pages Directory - Prototype HTML Files

This directory contains static HTML prototype files that were used during the initial design phase.
These files are NOT part of the active application.

**Active Application Structure:**
- Backend: Node.js + Express (see server.js)
- Views: EJS templates in /views/ directory
- Routes: Defined in /routes/ directory

**Prototype Files (Not Active):**
These HTML files served as design references and should not be modified or deployed:
- auth/*.html - Early auth page mockups
- products/*.html - Product browsing prototypes
- user/*.html - User dashboard mockups
- admin/*.html - Admin panel prototypes
- etc.

**For Development:**
To work on the active application, modify files in:
- /views/ for templates
- /routes/ for routing
- /controllers/ for business logic
- /public/ for static assets
